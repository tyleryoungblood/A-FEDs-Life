FEd standards

  • HTML5/CSS3 compliance
  • Make it polished (the little details, browser compatibility, accuracy to the original design, etc)
  • Make it fast (compress images, load only what's needed, etc)
  • Make it user friendly (testing with real users, making adjustments when it's obvious people aren't understanding how to use it)
